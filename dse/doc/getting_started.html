Getting Started¶

First, make sure you have the dse extension properly installed.

Connecting to Cassandra¶

The DSE extension builds on the core DataStax Cassandra Driver. Using the DSE extension is as simple as importing Cluster from the dse package instead of the core cassadnra package:

from dse.cluster import Cluster

cluster = Cluster()
session = cluster.connect()
session.execute("SELECT * FROM system.local")

These types extend the core API, but behave exactly like the core driver counterparts for the purpose of CQL execution (core documentation here).
